## How to setup

1. Fork this repository
2. Clone the forked repository
3. Install dependencies

   ```bash
   pnpm install
   ```

4. Build all projects (for **1st time** only)

   ```bash
   pnpm all
   ```

5. Run the development server

   ```bash
   pnpm run dev --filter program
   ```
