# `@codinasion/ascii-to-binary`

Convert ASCII to binary number system.

## Installation

```bash
# if you're using pnpm
pnpm add @codinasion/ascii-to-binary

# or, if you're using npm
npm install @codinasion/ascii-to-binary

# or, if you're using yarn
yarn add @codinasion/ascii-to-binary
```

## Usage

```javascript
import { AsciiToBinary } from "@codinasion/ascii-to-binary";

const ascii = "Hello World";

const binary = AsciiToBinary(ascii);
console.log(binary);

// Output
// 0100 1000 0110 0101 0110 1100 0110 1100 0110 1111 0010
// 0000 0101 0111 0110 1111 0111 0010 0110 1100 0110 0100
```
